# Investigación sobre sistemas Operativos Linux
## Debian 
#### Historia
Debian fue fundado por Ian Murdock en 1993, siendo una de las distribuciones Linux más antiguas. Su nombre proviene de la combinación del nombre de su entonces novia (ahora esposa) Debra y el suyo propio (Murdock, 1994). (/bir)
#### ¿Que es?
Debian es un sistema operativo libre y de código abierto que se caracteriza por su estabilidad y compromiso con los principios del software libre.(/bir)
#### Caracteristicas
+ Sistema de gestion de paquetes APT
+ Estabilidad como prioridad principal
+ Amplia arquitectura de hardware soportada
+ Ciclos de lanzamiento conservadores
#### Uso y aplicaciones
Ideal para servidores y sistemas críticos donde la estabilidad es fundamental. También sirve como base para muchas otras distribuciones.(/bir)
## Arch Linux
#### Historia
Creado por Judd Vinet en 2002, inspirado en CRUX, otra distribución Linux minimalista (Vinet, 2002). (/bir)
#### ¿Que es?
Una distribución rolling release que sigue el principio KISS (Keep It Simple, Stupid).(/bir)
#### Caracteristicas 
+ Rolling release (actualizaciones continuas)
+ Sistema de paquetes Pacman
+ Arch User Repository (AUR)
+ Enfoque minimalista y personalizable
#### Uso y aplicaciones
Popular entre usuarios avanzados que desean control total sobre su sistema. Usado en estaciones de trabajo de desarrolladores.(/bir)
## Rocky Linux
#### Historia
Creado en 2020 por Gregory Kurtzer como un reemplazo community-driven de CentOS después de cambios en la estrategia de Red Hat (Kurtzer, 2020).(/bir)
#### ¿Que es?
Una distribución enterprise-level diseñada para ser 100% compatible con Red Hat Enterprise Linux (RHEL).(/bir)
#### Caracteristicas
+ Compatibilidad binaria con RHEL
+ Enfoque en estabilidad empresarial
+ Desarrollo comunitario
+ Ciclos de soporte largos
#### Uso y aplicaciones
Infraestructura empresarial, servidores corporativos, y entornos donde se requiere compatibilidad con RHEL.(/bir)
## Garuda Linux
#### Historia
Distribución relativamente nueva, lanzada en 2020, basada en Arch Linux.(/bir)
#### ¿Que es?
Una distribución orientada al usuario final con enfoque en la facilidad de uso y apariencia visual.(/bir)
#### Caracteristicas
+ Interfaz gráfica moderna y atractiva
+ Herramientas de administración gráficas
+ Basado en Arch Linux
+ Optimizado para rendimiento
#### Uso y aplicaciones 
Estaciones de trabajo de usuarios finales, gaming, y multimedia.(/bir)
## Fedora
#### Historia
Patrocinado por Red Hat, Fedora comenzó en 2003 como continuación del proyecto Red Hat Linux (Red Hat, 2003).(/bir)
#### ¿Que es?
Una distribución community-driven que sirve como plataforma de prueba para tecnologías que eventualmente pueden incorporarse a RHEL.(/bir)
#### Caracteristicas
+ Lanzamientos frecuentes (cada 6 meses)
+ Enfoque en tecnologías innovadoras
+ Integración estrecha con tecnologías Red Hat
+ Strong focus on security
#### Uso y aplicaciones
Desarrollo de software, estaciones de trabajo de desarrolladores, y entornos académicos.(/bir)
## Manjaro
#### Historia
Basado en Arch Linux, Manjaro fue lanzado en 2011 por Roland Singer.(/bir)
#### ¿Que es?
Una distribución user-friendly que mantiene la filosofía de Arch pero con mayor facilidad de uso.(/bir)
#### Caracteristicas
+ Instalador gráfico fácil de usar
+ Rolling release con estabilidad testing
+ Gestor de controladores hardware integrado
+ Múltiples ediciones oficiales
#### Uso y aplicaciones
Usuarios que quieren beneficios de Arch Linux sin la complejidad de instalación y configuración.(/bir)
## CentOS
#### Historia
Originalmente lanzado en 2004, CentOS era un clon binario de RHEL. En 2020, Red Hat anunció cambios significativos en el proyecto.(/bir)
#### ¿Que es?
Históricamente fue un clon enterprise-level de RHEL, ahora transformado en CentOS Stream.(/bir)
#### Caracteristicas
+ (Históricamente) Clon binario de RHEL
+ Estabilidad enterprise-grade
+ (Actualmente) Versión rolling de RHEL
#### Uso y aplicaciones
Servidores web, infraestructura empresarial, y entornos donde se requiere estabilidad probada.(/bir)
## Kali Linux
#### Historia 
Sucesor de BackTrack Linux, Kali fue lanzado en 2013 por Offensive Security.(/bir)
#### ¿Que es?
Una distribución especializada en pruebas de penetración y seguridad informática.(/bir)
#### Caracteristicas
+ Más de 600 herramientas de seguridad preinstaladas
+ Entorno personalizado para auditorías de seguridad
+ Soporte para múltiples dispositivos y plataformas
+ Desarrollado específicamente para profesionales de seguridad
#### Uso y aplicaciones
Pruebas de penetración, auditorías de seguridad, forense digital, e investigación de seguridad.(/bir)
## Linux Mint
#### Historia
Lanzado en 2006 por Clement Lefebvre, inicialmente basado en Kubuntu, ahora basado en Ubuntu.(/bir)
#### ¿Que es?
Una distribución orientada a la facilidad de uso y experiencia de usuario para escritorio.(/bir)
#### Caracteristicas
+ Interfaz familiar para usuarios de Windows
+ Software multimedia incluido por defecto
+ Herramientas de administración gráficas
+ Estabilidad y facilidad de uso
#### Uso y aplicaciones
Escritorio para usuarios domésticos, oficinas, y educación.(/bir)
## Ubuntu 
#### Historia
Creado por Mark Shuttleworth y Canonical Ltd., lanzado por primera vez en 2004.(/bir)
#### ¿Que es?
Una de las distribuciones Linux más populares, especialmente para escritorio, basada en Debian.(/bir)
#### Caracteristicas
+ Lanzamientos regulares cada 6 meses
+ Versiones LTS con soporte extendido
+ Gran comunidad y soporte comercial
+ Enfoque en facilidad de uso
#### Uso y aplicaciones 
Escritorio para usuarios principiantes, servidores, cloud computing, y IoT.(/bir)
## Alpine Linux
#### Historia
Desarrollado desde 2005, Alpine se centra en la seguridad y minimalismo.(/bir)
#### ¿Que es?
Una distribución diseñada para seguridad, simplicidad y eficiencia de recursos.(/bir)
#### Caracteristicas
+ Basado en musl libc y BusyBox
+ Extremadamente ligero
+ Enfoque en seguridad
+ Popular para contenedores Docker
#### Uso y aplicaciones 
Contenedores Docker, routers, sistemas embebidos, y aplicaciones donde la seguridad es crítica.(/bir)
## AlmaLinux
#### Historia
Creado en 2021 por CloudLinux Inc. como reemplazo de CentOS después de los cambios en la estrategia de Red Hat.(/bir)
#### ¿Que es?
Un clon binario de RHEL desarrollado por la comunidad, similar al CentOS original.(/bir)
#### Caracteristicas
+ Compatibilidad 1:1 con RHEL
+ Desarrollo comunitario
+ Soporte a largo plazo
+ Transición fácil desde CentOS
#### Uso y aplicaciones
Servidores empresariales, migraciones desde CentOS, infraestructura crítica.(/bir)
